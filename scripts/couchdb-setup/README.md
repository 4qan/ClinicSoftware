# CouchDB Setup

One-time setup for syncing Clinic Software between doctor and nurse machines over LAN.

**Full installation guide:** [INSTALLATION-GUIDE.md](INSTALLATION-GUIDE.md)

## Files

| File | Purpose |
|------|---------|
| `install-couchdb.ps1` | Installs and configures CouchDB on the doctor's Windows machine |
| `uninstall-couchdb.ps1` | Removes CouchDB completely (for clean reinstall) |
| `local.ini` | CouchDB config template (used by install script) |
| `INSTALLATION-GUIDE.md` | Step-by-step setup guide for both machines |
| `ClaudeCode-Comms.md` | Cross-machine debug log (development only) |
