# Requirements: ClinicSoftware

**Defined:** 2026-03-19
**Core Value:** The doctor can see a patient, write a prescription with medication autocomplete, and print it in under 2 minutes, even with no internet.

## v1.4 Requirements

Requirements for v1.4 Slip Assignment & Print Settings. Each maps to roadmap phases.

### Slip Assignment

- [ ] **SLIP-01**: User can designate each medication as "dispensary" or "prescription" when adding it
- [ ] **SLIP-02**: New medications default to "dispensary"
- [ ] **SLIP-03**: Prescription slip only prints medications tagged as "prescription"
- [ ] **SLIP-04**: Dispensary slip only prints medications tagged as "dispensary"
- [ ] **SLIP-05**: Slip assignment is stored with the medication snapshot (persists with the encounter)

### Print Settings

- [ ] **PRSET-05**: User can toggle auto-print on/off in Print Management settings
- [ ] **PRSET-06**: Auto-print preference persists across sessions

## Future Requirements

None currently deferred.

## Out of Scope

| Feature | Reason |
|---------|--------|
| Both-slips option per medication | Doctor confirmed each med is one slip or the other, never both |
| Per-visit print settings | Global settings sufficient for single-doctor clinic |
