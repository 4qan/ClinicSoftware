# Plan 04-01: Font Installation & Print CSS

## Objective
Install `@fontsource-variable/noto-nastaliq-urdu`, import it in the app entry point so it is eagerly loaded and service-worker cached, and define the `.urdu-cell` print CSS class for Phase 5 consumption.

## must_haves
- [ ] Noto Nastaliq Urdu variable font is installed and imported at app startup
- [ ] `.urdu-cell` class exists in print styles with Nastaliq font, adequate line-height, and overflow visible
- [ ] Font woff2 file is present in build output (service worker will precache it via existing globPatterns)

<tasks>

<task id="04-01-01" title="Install font package and add import">
<action>
1. Install the font package:
```bash
cd /Users/furqantariq/Documents/Projects/ClinicSoftware && npm install @fontsource-variable/noto-nastaliq-urdu
```

2. Add the font import to `src/main.tsx`, before the CSS import:
```typescript
import '@fontsource-variable/noto-nastaliq-urdu'
```

The file should read:
```typescript
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import '@fontsource-variable/noto-nastaliq-urdu'
import './index.css'
import App from './App.tsx'

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <App />
  </StrictMode>,
)
```
</action>

</task>

<task id="04-01-02" title="Add .urdu-cell print CSS class">
<action>
Add the `.urdu-cell` class inside the existing `@media print { ... }` block in `src/index.css`, after the last existing rule (the `.prescription-slip, .dispensary-slip` rule):

```css
  /* Urdu Nastaliq rendering for prescription cells */
  .urdu-cell {
    font-family: 'Noto Nastaliq Urdu Variable', serif;
    direction: rtl;
    line-height: 2.2;
    overflow: visible;
    padding-top: 4px;
    padding-bottom: 4px;
  }
```

This goes inside the `@media print` block, before its closing `}`.
</action>
<verify type="automated">
```bash
cd /Users/furqantariq/Documents/Projects/ClinicSoftware && grep -A5 'urdu-cell' src/index.css | grep -q 'Noto Nastaliq Urdu Variable' && echo "PASS: .urdu-cell CSS defined"
```
</verify>
</task>
