## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| Browser тЖФ localStorage | Solo deployment persists `deploymentMode`, `couchUrl`, `soloCredentials` (PBKDF2 envelope), `clinic_couch_session` | Mode flag, hash + salt + iterations, session payload `{username, role}` |
| Browser тЖФ IndexedDB (PouchDB) | All clinical data (patients, visits, medications) | PHI, fully local in solo mode |
| Solo provider tree тЖФ Networked provider tree | Mutually exclusive at boot via `getDeploymentMode()` switch in `App.tsx` | None тАФ provider trees never co-mount |
| Solo runtime тЖФ Network | No outbound traffic in solo mode (D-17). Verified by `installFetchSpy()` regression net | None (zero CouchDB-bound fetches) |
