# Plan 01: Urdu Columns, Bilingual Headers & Section Labels

## Context
Modify PrescriptionSlip.tsx and DispensarySlip.tsx to render Form, Dosage, Frequency, and Duration in Urdu (via `toUrdu()`), add bilingual column headers (English + Urdu) on all 7 medication columns, and add bilingual section labels on the prescription slip. Uses the `.urdu-cell` CSS class and `toUrdu()` helper from Phase 4.

## must_haves
- [ ] Form, Dosage, Frequency, Duration columns render Urdu text (Nastaliq script) on both print slips
- [ ] All 7 medication column headers display bilingual labels (English on top, Urdu below)
- [ ] RTL text in Urdu cells coexists with LTR English drug names without layout breakage
- [ ] "Clinical Notes" and "Instructions" section headers on prescription slip show bilingual labels
- [ ] Brand Name, Salt, Strength columns remain English-only
- [ ] Fallback: untranslated values render as original English without visual breakage

<task id="05-01-01" title="Add Urdu columns to PrescriptionSlip">
<description>
In `src/components/PrescriptionSlip.tsx`:

1. Import `toUrdu` from `@/constants/translations`.
2. Add bilingual headers to all 7 medication columns (not #). Each `<th>` gets a stacked layout: English text on top, then `<br />`, then `<span dir="rtl" className="urdu-cell" style={{ fontSize: '9pt' }}>` with the Urdu label. Urdu labels per context decisions: Brand Name/دوا, Salt/نمک, Strength/طاقت, Form/شکل, Dosage/خوراک, Freq/تعداد, Duration/دورانیہ.
3. For body `<td>` cells of Form, Dosage, Frequency, Duration: add `dir="rtl"` attribute and `urdu-cell` class, wrap value with `toUrdu()`. Keep existing padding classes.
4. Brand Name, Salt, Strength `<td>` cells remain unchanged.
5. Update "Clinical Notes" `<h3>` to: `Clinical Notes / <span dir="rtl" className="urdu-cell">طبی نوٹس</span>`
6. Update "Instructions" `<h3>` to: `Instructions / <span dir="rtl" className="urdu-cell">ہدایات</span>`
7. Patient info labels (Patient, ID, Age/Gender, Date) stay English-only.
</description>
