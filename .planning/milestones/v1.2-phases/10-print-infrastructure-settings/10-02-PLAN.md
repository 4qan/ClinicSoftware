  <action>
    Modify `src/pages/PrintVisitPage.tsx`:

    1. **Import print settings**: Add imports for `getPrintSettings`, `PAPER_SIZES`, `calcMargin` from `@/db/printSettings`. Import `PrintSettings` type.

    2. **Load print settings on mount**: Add `printSettings` state (type `PrintSettings | null`). In the existing loadData useEffect, add `getPrintSettings()` to the Promise.all alongside getPatient and getClinicInfo. Store result in state.

    3. **Add injectPageStyle/removePageStyle helpers** (defined inside the component or as module-level functions):
       ```
       function injectPageStyle(size: PaperSize, margin: number): void
         - Remove existing element with id 'print-page-style' if present
         - Create <style> element with id 'print-page-style'
         - Set textContent to `@page { size: ${width}mm ${height}mm portrait; margin: ${margin}mm; }`
         - Append to document.head

       function removePageStyle(): void
         - Remove element with id 'print-page-style' if it exists
       ```

    4. **Update handlePrint**: Read the active size from printSettings (prescription or dispensary based on mode). Call injectPageStyle with the size and calcMargin result. Then set printMode and setTimeout for window.print() as before.

    5. **Update auto-print useEffect**: Same pattern -- inject page style before the setTimeout print call. Read size from printSettings state (guard against null).

    6. **Update handleAfterPrint**: Call removePageStyle() in addition to setPrintMode(null).

    7. **Replace CSS class toggle with conditional rendering** (PRENG-03):
       Replace the two div wrappers with:
       ```tsx
       const showPrescription = printMode !== null ? printMode === 'prescription' : previewMode === 'prescription'
       const showDispensary = printMode !== null ? printMode === 'dispensary' : previewMode === 'dispensary'

       {showPrescription && <PrescriptionSlip ... />}
       {showDispensary && <DispensarySlip ... />}
       ```
       Remove wrapping divs with `print-hidden` and `hidden` classes entirely.

    8. **Add size badge** near the print button (inside the no-print toolbar area):
       ```tsx
       <span className="text-xs text-gray-500 bg-gray-100 border border-gray-200 rounded px-2 py-1">
         Paper: {PAPER_SIZES[activeSize].label}
       </span>
       ```
       Where `activeSize` is derived from printSettings based on current previewMode. If printSettings is null (still loading), show nothing or default label.

    Modify `src/index.css`:
    - **Remove** line 30: `@page { size: A5 portrait; margin: 8mm; }` -- this is now handled by dynamic injection.
    - Keep the `.print-hidden` class definition (other code may still reference it, and removing unused CSS is harmless but safe to leave). Actually, since PRENG-03 replaces CSS hiding with conditional rendering, the `.print-hidden` class is no longer used by PrintVisitPage. Leave it for now (safe), but add a comment: `/* Legacy: may be removed after Phase 10 verification */`.

    **Anti-patterns to avoid:**
    - Do NOT use CSS custom properties in @page (they don't work per research)
    - Do NOT inject style in useEffect (must be synchronous before window.print call)
    - Do NOT seed DB on first load (read-time fallback to A5 per user decision)
    - Do NOT remove the setTimeout before window.print() (needed for React re-render)
