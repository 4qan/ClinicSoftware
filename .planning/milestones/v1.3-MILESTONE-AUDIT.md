## Root Cause Analysis

Both gaps (FOCUS-01 partial, FOCUS-02 unsatisfied) share a single root cause:

Plan 12-01 referenced `src/pages/LoginPage.tsx` and `src/pages/ChangePassword.tsx`, but both files actually live at `src/auth/LoginPage.tsx` and `src/auth/ChangePassword.tsx`. The cleanup was applied to the wrong paths (which don't exist), so the legacy `focus:ring-*` classes were never stripped from 9 className strings across both auth files.

**Fix:** Strip `focus:ring-2 focus:ring-blue-500 focus:border-blue-500 focus:ring-offset-2` from both src/auth/ files and extend the class audit test to scan src/auth/.
